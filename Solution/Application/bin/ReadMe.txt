Notes: It is required to install SSIS before running the conversion tool.


1. Open the configuration file "iTrak.ImporterMain.exe.config" and modify the connection string "iTrak" pointing to the proper server name

2. Modify the connection string "Source" pointing to the proper server and source database

3. Change the property and department name in the xml file PropertyMappings.xml to reflect the name in iTrak. Do not change the property code

4. Start to run the conversion application
